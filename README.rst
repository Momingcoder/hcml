hcml
===========

A plugin for [Hachi](https://github.com/guokr/Hachi) to detect meaningless sentences.

Usage
------

::

    > m = Meaningless()
    > m.predict(message)

Dependence
------------

* jieba

Attention
-----------

You need to provide English Words in `./data/` .

Of course you can simply copy from this package.
